# T-Mobile Feedback AI - Frontend

React + TypeScript + TailwindCSS frontend application.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## Tech Stack

- React 18
- TypeScript
- TailwindCSS
- Inter/Poppins fonts
