# Font Files

Place the following font files in this directory:

## Lavishly Yours
- `LavishlyYours-Regular.woff2` (preferred)
- `LavishlyYours-Regular.woff` (fallback)
- `LavishlyYours-Regular.ttf` (fallback)

## Knewave
- `Knewave-Regular.woff2` (preferred)
- `Knewave-Regular.woff` (fallback)
- `Knewave-Regular.ttf` (fallback)

The fonts are configured in `src/index.css` with @font-face declarations and in `tailwind.config.js` as `font-amsterdam` (Lavishly Yours) and `font-avallon` (Knewave) utility classes.
